Tuning Filter Quality
To raise stop-band attenuation beyond 80 dB, increase FIR_MAX_K in config.h:

K=16 → ~80 dB   (default, ~12% Core 0)
K=24 → ~100 dB  (~18% Core 0)
K=32 → ~120 dB  (~24% Core 0)

Also adjust FIR_KAISER_BETA to match (β=9.0 for 100 dB). ESP32-S3 has headroom for K=32 without audio dropouts.
